    LEARNING
    Cxx and More
    The X source code transpiles to human readable Cxx code. A familiar environment for Cxx developers, a comfortable start for new developers.
    Trust and Compliance
    The generated Cxx code is code that works everywhere and is compatible with all Cxx stuff. Develop Cxx in a simpler and more readable way.
    Simplicity
    More modern, readable and comfortable than Cxx syntax. Simple and maintainable code, simplicity and functionality.
    Pretty Transpile

    X produces Cxx code that's great, readable, and pretty neat. You don't need to waste time editing the transpiled code.

    • Adds information about the transpilation
    • Human readable
    • Splits the code into regions
    • Groups codes by rearranging them
    • Single output file
    Documenter

    The X compiler has a documenter internally. It makes clear documentation for your code.

    • JSON format
    • Human readable
    • Defines only
    • Single output file per file
    • Easy to process
